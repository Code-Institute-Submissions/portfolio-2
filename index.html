Website Info


Just Another Social Network

Primarily built on Django - The website offers a number of features similar to most Social Networks and a number of "Niche" features specific to the website. The website is a social Network for Competitive Gamers and we allow users to make "Character Profiles". Each Character Profile can add specific attributes based on the game that they play. While some games require only one attribute, others are more complex and may require 3 or more, the design of these "Attributes" allow flexibitly without creating different object - The website also has a flexible searching system for game characters that your would like to play with, which have desirable attributes and ranking.

Data Visualisation

Cut through explanations and get to the heart of analysing data quickly using interactive charts and statistics. Whatever the Database, there is a way to represent it quickly.
